# NgobrolApp

NgobrolApp ini dibuat menggunakan flutter, masuk menggunakan akun gmail anda dan temukan teman untuk ngobrol disaat kalian luang, pusing, capek dsb deh pokoknya aplikasi ini buat ngobrol...
